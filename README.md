# Brix Core [![Build Status](https://travis-ci.org/thx/brix-core.png?branch=master)](https://travis-ci.org/thx/brix-core)

## 演示

安装开发环境，启动 HTTP 服务。

```
$ npm install
$ npm start
```

访问 <http://localhost:5000/demo/>
